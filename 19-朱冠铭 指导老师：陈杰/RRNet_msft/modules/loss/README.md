## Introduction

This is Focalloss for RetinaNet

$FL= - \alpha_{t} * \left( 1-P_{t} \right)^{\gamma}*CE(P_{t}) $

## Function

- **\_\_init\_\_()**
  - **params**:
    - **alpha** : a float32 number for hyperparameter
    - **gamma** : a float32 number for hyperparameter
    - **classnum** : a int number for amount of classes
